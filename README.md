
# WzryLibrary
IOS项目：王者荣耀手册
## 功能介绍
1. 查看王者荣耀的英雄信息
2. 查看王者荣耀的装备信息
3. 查看王者荣耀的铭文信息
4. 查看王者荣耀的召唤师技能信息
## 所用技术
1. tableview的基本使用
2. 多线程加载图片
3. nevigation的使用
4. 毛玻璃效果的实现
5. 通过url获取图片内容
6. wkwebview的使用基础
7. collectionview的使用基础
8. 线程传值使用nsdictionary传多个值
## 所用资源
1. 王者荣耀英雄列表
2. 王者荣耀铭文列表
3. 王者荣耀装备列表
4. 王者荣耀召唤师技能列表
5. 图片信息
6. 王者荣耀官网链接
